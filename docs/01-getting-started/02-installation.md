# Installation

## Prerequisites

Before installing AppPulse, ensure your system meets the following requirements:

- PHP 8.2, 8.3, or 8.4
- Laravel 10.x, 11.x, or 12.x
- Composer installed

## Install via Composer

Install the package using Composer:

```bash
composer require cleaniquecoders/app-pulse
```

The package will automatically register itself with Laravel's package discovery.

## Verify Installation

After installation, you can verify that the package is installed correctly by checking:

```bash
composer show cleaniquecoders/app-pulse
```

This should display package information including version and dependencies.

## Next Steps

Once installed, proceed to [Configuration](03-configuration.md) to set up the package in your application.
